# 🔍 가짜뉴스 탐정단

뉴스의 진위를 판단하는 교실용 경제·미디어 리터러시 게임

-----

## 게임 개요

학생들이 제시된 뉴스를 보고 진짜/가짜 여부를 판단하는 게임입니다.  
두 가지 모드로 진행할 수 있습니다.

### 🏛️ 경매 모드

- 진짜라고 생각하는 뉴스를 경매로 구매
- **진짜 뉴스** 구매 → 투자금 × 2 수익
- **가짜 뉴스** 구매 → 투자금 손실

### 📈 투자 모드

- 각 뉴스에 대해 진짜/가짜 방향을 선택하고 원하는 금액만큼 투자
- **정답** 방향 투자 → 투자금 × 2 수익
- **오답** 방향 투자 → 투자금 손실

-----

## 게임 흐름

```
1. 홈 화면  →  모드 선택 (경매 / 투자)
2. 관리자 패널  →  플레이어 설정 + 뉴스 카드 등록 + 진위 표시
3. 탐색 단계  →  모든 뉴스를 순서대로 미리 확인
4. 게임 단계  →  뉴스별 경매 또는 투자 진행
5. 결과 발표  →  뉴스별 정답 공개 → 최종 순위 발표
```

-----

## 교사 사용 가이드

### 1단계 — 뉴스 카드 등록

1. **관리자 패널 → 뉴스 카드 탭**에서 이미지 업로드
1. 각 뉴스에 **진짜 / 가짜** 정답 설정 (필수)
1. 제목 입력 선택사항

### 2단계 — 플레이어 설정

- 플레이어 탭에서 모둠/팀 이름 변경
- 초기 잔액 조정 (기본: 1,000,000원)
- 1~8명(팀)까지 설정 가능

### 3단계 — 게임 진행

#### 경매 모드 (교사 조작)

|버튼          |기능                                   |
|------------|-------------------------------------|
|**입찰자 선택**  |입찰 의사가 있는 플레이어 버튼 클릭 (토글)            |
|**📈 가격 올리기**|설정된 단위만큼 가격 인상 (입찰자 초기화됨)            |
|**🔨 낙찰!**   |현재 입찰자에게 낙찰 (1명이면 자동 / 여러 명이면 교사가 선택)|
|**유찰 (패스)** |해당 뉴스 유찰 처리                          |

#### 투자 모드 (플레이어 조작)

1. 각 플레이어가 진짜/가짜 버튼 선택
1. 투자 금액 입력 (0원 또는 패스 가능)
1. 확인 클릭 → 잔액에서 즉시 차감
1. 모든 플레이어 완료 시 “다음 뉴스” 버튼 활성화

## 기술 메모

- **저장**: 게임 상태와 뉴스 이미지는 `localStorage`에 base64로 저장
  - 이미지가 많거나 클 경우 5MB 제한에 주의
  - 매 수업마다 새로 설정하는 용도에 적합
- **의존성**: 외부 라이브러리 없음 (구글 폰트 제외)
- **브라우저**: Chrome, Edge, Safari, Firefox 최신 버전 지원
